Questions: Discuss the scope of var, let, and const ?

Definition: var is a global scope. let and const is a block scope.

var variable Can be accessed from anywhere in the code.It can be accessed from outside and inside the block scope. Variables declared with var are function-scoped, meaning they are accessible only within the function in which they are declared or, if declared outside of any function, they become global variables.

let and const block element.A variable declared with let and const is block scoped.If let and const variables are declared inside a block scope, they cannot be accessed from outside the block scope.

const: A value declared with const cannot be redeclared
